bower

Manage CSS/JS dependencies and vendor packages.


# install bower globslly
npm install -g bower

# Search for a dependency in the Bower registry.
$ bower search <query>

# Install a dependency. Optionally specify the version
$ bower install <package> --save-dev
$ bower install <package>#<version> --save-dev

# Uninstall a dependency.
$ bower uninstall <package> --save-dev

# Update a dependency to the latest version available.
$ bower update <dep>
