yeoman

Yeoman (yo) is a scaffolding tool that builds out initial app structure based on defined generators (templates).

Generators typically include:

  • files
  • directories
  • package requirements
  • tasks (build, test)
  • version control repo

# Install yeoman globally
$ npm install -g yo


# Install a generator or two
$ npm install -g generator-webapp
$ npm install -g gulp-webapp
$ npm install -g generator-zf5
$ npm install -g generator-gulp-foundation


# Start a project and get going
$ mkdir my-project && cd my-project
$ yo zf5

Generators are opinionated.
